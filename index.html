Simulador de Tarjeta de Crédito - Davivienda

Objetivo

Este simulador fue creado para agilizar el proceso de simulación de intereses de tarjetas de crédito durante las consultas del Call Center de Davivienda. Permite a los asesores generar cálculos rápidos y precisos de cuotas e intereses, proporcionando una mejor experiencia al cliente.

Funcionamiento

El simulador toma como base la tasa efectiva anual (E.A.), el número de cuotas, y el valor de la transacción para calcular los intereses que el cliente debe pagar en cada cuota. El sistema también gestiona las fechas clave como la fecha de transacción, la fecha de corte (solo permite seleccionar viernes o penúltimo día del mes) y la fecha de pago, que se genera automáticamente según los datos proporcionados. Los cálculos incluyen el reparto de intereses y el abono a capital, mostrando claramente el saldo y las cuotas mes a mes.
